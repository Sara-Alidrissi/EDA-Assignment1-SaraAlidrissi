Dataset Description:

Dataset Name: Sample Superstore Dataset
Source: https://www.kaggle.com/datasets/bravehart101/sample-supermarket-dataset
Number of Rows: 9994
Number of Columns: 13

Features (Columns):
- Ship Mode: shipping method
- Segment: customer segment
- Country: country
- City: city
- State: state
- Postal Code: postal code
- Region: region
- Category: product category
- Sub-Category: product type
- Sales: revenue
- Quantity: number of items sold
- Discount: discount applied
- Profit: profit

Purpose: The purpose of this dataset is to analyze sales performance, understand the relationship between variables like sales, quantity, and profit, and identify trends and patterns.
